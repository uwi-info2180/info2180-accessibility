Tables

Use tables for tabular data and not for layouts. Ensure you use appropriate tags such as caption, thead, tbody, tr, td, th, etc…

Forms

It is good practice to use a label for each of your for input fields. You can also add reference an additional description using the aria-labelledby and aria-describedby attributes. State as well the fields that are required.

Links

We must ensure that links are easy to identify since the colour stands out. Also ensure that links can be understood even if not in the context of the words around it.

So don’t say, For more information on device independence click here. Instead say something like Read more about device independence.

Images
